C++ Solution:
class Solution {
    void traverse(vector<int>&res,TreeNode*node)
    {
        if(node==nullptr)return;
        traverse(res,node->left);
        res.push_back(node->val);
        traverse(res,node->right);
    }
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int>result;
        traverse(result,root);
        return result;
    }
};

JAVA SOLUTION:
class Solution {
    public void traverse(TreeNode root,List<Integer>res)
    {
        if(root==null)return;
        traverse(root.left,res);
        res.add(root.val);
        traverse(root.right,res);
    }
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer>res=new ArrayList<>();
        traverse(root,res);
        return res;
    }
}

You just tell the computer what to do for one node, and it handles the rest.C++void traverse(TreeNode* node, vector<int>& res) {
    // Stop if we hit a dead end (null node)
    if (node == nullptr) return; 

    // Step 1: LEFT
    traverse(node->left, res);  
    
    // Step 2: NODE
    res.push_back(node->val);   
    
    // Step 3: RIGHT
    traverse(node->right, res); 
}
To implement this easily:Always write your base case first (if (node == nullptr) return;). This stops the function from running forever.Write the three steps in order: Left recursive call, process the current node's value, Right recursive call.In your main inorderTraversal function, just create the vector, pass it into your helper function, and return it when it's done!
